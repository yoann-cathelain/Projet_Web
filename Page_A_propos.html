Projet de WEB Année Spéciale 2021-2022

Le but de ce projet est de réaliser un "chassis" de site web comportant plusieurs pages d'un site type de vente, on y trouve les pages suivantes :

  1. Une d'accueil permettant d'attirer le consommateur
  2. Une page de recherche permettant de rechercher n'importe quel type de produit
  3. Une détaillant un produit permettant de mieux voir ses caractéristiques
  4. Une page simulant un panier ayant comme fonctionnalité d'ajouter un article, d'en supprimer et de voir les articles présent à l'intérieur
  5. Une page à propos décrivant le projet, précisant la nature du projet
  6. Une page description technique expliquant les fonctionnalités principales du code

Il est également précisé que le site en question doit être responsive et donc adapter à plusieurs taille d'écran
